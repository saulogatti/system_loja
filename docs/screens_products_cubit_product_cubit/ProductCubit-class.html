ProductCubit class

Gerencia o estado da tela de produtos e operações com a base de dados.

Este Cubit é responsável por coordenar operações CRUD (criar, ler, atualizar, deletar) de produtos, mantendo o estado sincronizado com a IProductRepository. Emite estados que refletem o resultado de cada operação para a UI.
