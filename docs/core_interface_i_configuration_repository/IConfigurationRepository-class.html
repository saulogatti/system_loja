IConfigurationRepository

IConfigurationRepository class abstract interface

Interface que define o contrato para operações de configuração do sistema.

Esta interface gerencia as configurações globais da aplicação, incluindo backup/restore, limpeza de dados e reset de configurações.

Todas as operações retornam ResultStatus com AppSettings em sucesso e mensagem de erro em texto — sem propagar exceções para a UI.
