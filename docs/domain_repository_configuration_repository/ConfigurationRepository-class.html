ConfigurationRepository

ConfigurationRepository class

Repositório para gerenciamento de configurações da aplicação.

Persiste e recupera AppSettings via CacheManager e aplica o tema (cor primária e modo escuro) em tempo real via ISettingsService. Todos os erros são capturados internamente e devolvidos como ResultStatus.error com mensagem amigável.
