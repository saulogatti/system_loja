CustomerRepository

CustomerRepository class

Repositório para gerenciamento de clientes usando Drift.

Coordena operações CRUD sobre Customer via CustomerDao e registra eventos de auditoria via ILogRepository. Todos os erros são capturados internamente e devolvidos como ResultStatus.error com mensagem amigável.
