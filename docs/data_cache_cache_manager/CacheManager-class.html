CacheManager

CacheManager class

Gerencia o cache de dados da aplicação.

Utiliza o padrão singleton para garantir uma única instância de CacheManager durante o ciclo de vida da aplicação.

O cache é persistido em arquivos JSON no diretório de documentos da aplicação, organizados por tipo de objeto.

Exemplo de uso:

final cache = CacheManager.instance;
await cache.ensureInitialized();

// Armazenar um objeto
await cache.set(meuObjeto);

// Recuperar um objeto
final objeto = await cache.get<MinhaClasse>('chave', MinhaClasse.fromJson);
