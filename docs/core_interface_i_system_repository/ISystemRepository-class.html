ISystemRepository

ISystemRepository class abstract interface

Interface que define o contrato para operações de configuração do sistema.

Esta interface gerencia as configurações técnicas e de baixo nível do sistema, como parâmetros de inicialização, versões, e metadados.

Diferente de IConfigurationRepository que gerencia preferências do usuário, esta interface lida com configurações do próprio sistema.

Exemplo de uso:

final repository = appInjection.get<SystemRepository>();

final resultado = await repository.getSystemConfiguration();
resultado.when(
  onSuccess: (config) => print('Configuração carregada: ${config.id}'),
  onError: (erro) => print('Falha: $erro'),
);
