FileNameStringExtensions

FileNameStringExtensions extension

Extensões para manipulação segura de strings em nomes de arquivos.

Fornece métodos para sanitizar, normalizar e validar strings que serão usadas como nomes de arquivos, garantindo compatibilidade entre diferentes sistemas operacionais.
