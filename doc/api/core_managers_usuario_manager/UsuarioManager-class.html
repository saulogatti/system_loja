UsuarioManager

UsuarioManager class

Gerenciador de Usuários

Utiliza um mecanismo de sincronização para evitar condições de corrida e recarrega dados antes de salvar para prevenir perda de dados. Implementa funcionalidades de hash de senha para segurança.
