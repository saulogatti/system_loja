DatabaseHelper

DatabaseHelper class

Gerenciador do banco de dados SQLite

Implementa o padrão Singleton para garantir uma única instância de conexão com o banco de dados em toda a aplicação.

Responsável por:

  • Inicializar o banco de dados
  • Criar as tabelas e índices
  • Gerenciar migrações de versão
  • Fornecer acesso à instância do banco
