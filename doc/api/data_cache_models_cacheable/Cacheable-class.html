Cacheable

Cacheable class abstract

Define o objeto que pode ser armazenado em cache.

Esta interface define os métodos necessários para que um objeto possa ser serializado e armazenado no sistema de cache da aplicação.

Exemplo de implementação:

class MinhaClasse implements Cacheable {
  @override
  String get cacheKey => 'minha_classe_$id';

  @override
  Map<String, dynamic> toJson() => {'id': id};
}
